# 化学实验模拟与数据平台

## 简介
一个为化学专业同学设计的 Web 应用，提供实验记录管理、化学计算、滴定模拟、反应预测、报告生成等功能。

## 功能列表
- **实验记录管理**：增删改查个人实验数据
- **化学计算器**：摩尔质量、溶液配制、单位换算
- **滴定模拟器**：强酸强碱滴定曲线可视化
- **反应预测**：基于规则库的简单有机反应预测
- **实验教学库**：内置多个经典实验的详细步骤与原理
- **报告生成**：从实验记录生成可编辑报告并导出 PDF

## 技术栈
- 后端：Python + Flask
- 数据库：SQLite
- 前端：HTML5 + Bootstrap 5 + ECharts + Quill
- 部署：阿里云 ECS（推荐）

## 安装与运行
1. 克隆项目：`git clone https://github.com/你的用户名/chemistry-lab.git`
2. 进入目录，创建虚拟环境并激活
3. 安装依赖：`pip install -r requirements.txt`
4. 初始化数据库：`python init_db.py`
5. 运行：`python app.py`
6. 访问 `http://127.0.0.1:5000`
